# Hero Demo Runtime Artifacts

This directory contains curated, deterministic local simulation artifacts used by the Actenon README and hero demo.

These files are committed intentionally so reviewers can inspect example Receipt, Refusal, replay, and incident-simulation outputs without running the demo first.

They are not production secrets, live customer data, or generated runtime state from a real deployment.

New local runs may create additional files under `artifacts/`; those should not be committed unless they are intentionally curated examples.
