# Social Preview Brief

Purpose: GitHub social preview image for the public Actenon repository.

## Canvas

- Size: 1280 x 640 px.
- Format: PNG or JPG for GitHub social preview upload.
- Keep enough padding for crop-safe previews on GitHub, X/LinkedIn, and chat unfurls.

## Copy

Main text:

```text
No valid proof, no execution.
```

Subtitle:

```text
Open proof gate and receipt standard for consequential AI actions.
```

## Visual Direction

- Show a terminal-style refusal moment and receipt digest.
- Use the demo shape: agent attempts consequential action -> `REFUSED` -> `side_effect_executed: false` -> `artifact_digest: sha256:...`.
- Keep the visual legible at small sizes; one refusal line and one receipt digest are enough.
- Suggested supporting words: `ActionIntent`, `PCCB`, `Receipt`, `Refusal`.

## Avoid

Do not include or imply:

- industry standard
- insurer approved
- regulator approved
- certified
- formally verified
- unhackable
- prevents all prompt injection
- stops data leaks or filters model output
- prevents all incidents
- production KMS/HSM custody unless separately implemented and verified
- Actenon Network or Agent Trust Score as existing product claims
