# Trust Artifact V1 Vectors

Deterministic, public test vectors for:

- `issuer_status v1`
- `approval_artifact v1`

The keys are throwaway local fixtures. Only public keys are committed. The
vectors exercise good standing, revoked/stale/expired status, missing status,
signature tampering, exact-action approval, and action laundering.

These vectors define verification interoperability only. They do not contain
an issuer registry, approval service, signing service, or private key custody.
